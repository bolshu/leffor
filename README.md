# Leffor

**Leffor** (last effort) is a pet project for python learning.

## Development

### Start project

1. Make venv
```
-m venv ./.venv
```

1. Join to venv
```
source ./.venv/bin/activate
```

1. Install dependencies
```
pip install -r requirements.txt
```

1. Run server
```
python3 ./src/main.py
```

### Linting

To lint project run command
```
flake8 ./src
```
