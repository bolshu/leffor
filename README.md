# Leffor

**Leffor** (last effort) is a pet project for python learning.

## Development

### Start project

Make venv
```
-m venv ./.venv
```

Activate to venv
```
source ./.venv/bin/activate
```

Install dependencies
```
pip install -r requirements.txt
```

Run server
```
python3 ./src/main.py
```

### Linting

To lint project run command
```
flake8 ./src
```
